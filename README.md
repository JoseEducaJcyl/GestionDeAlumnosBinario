# Sistema de Gestión de Estudiantes - Serialización y Cálculo

## Descripción
Aplicación modular en Java que gestiona una lista de estudiantes, calcula estadísticas y utiliza serialización para guardar y recuperar los datos.

## Funcionalidades
- Crear estudiantes con nombre y nota
- Calcular la nota media de todos los estudiantes
- Guardar lista de estudiantes en archivo binario
- Leer estudiantes desde archivo binario
- Mostrar todos los estudiantes registrados

## Tecnologías
- Java
- Serialización (interfaz Serializable)
- ObjectOutputStream / ObjectInputStream
- ArrayList
- Clases y métodos estáticos
- Manejo de excepciones
